## 🚀 빠른 시작

### 1단계. Python 환경 준비 (3.9 이상 권장)

```bash
# 가상환경 생성 (권장)
python -m venv venv

# 활성화
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
```

### 2단계. 패키지 설치

```bash
pip install -r requirements.txt
```

> ⏱️ 설치 시간: 약 1~2분 (pykrx, streamlit 포함)

### 3단계. 앱 실행

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501` 이 열립니다.

## 📊 분석 로직 상세

### 재무 필터 (1차 필터)

| 항목 | 기본값 | 의미 |
|------|--------|------|
| PER | ≤ 15배 | 주가가 이익 대비 저평가 |
| PBR | ≤ 1.5배 | 주가가 자산 대비 저평가 |
| ROE | ≥ 10% | 자기자본 수익성 양호 |

> **ROE 계산**: pykrx는 ROE를 직접 제공하지 않으므로  
> `ROE = EPS ÷ BPS × 100`으로 계산합니다.

### 기술적 조건 (차트 점수, 40점)

```
✅ 현재가 > 20일 이동평균    → 단기 상승 추세
✅ 20일선 > 60일선 (정배열) → 중기 상승 추세
✅ 60일선 상승 중           → 장기 추세 확인
✅ 20일 신고가 돌파 (±1%)   → 모멘텀 신호
✅ 거래량 ≥ 20일 평균 × 1.5 → 매수세 유입
```

### 거래대금 단위

```
pykrx 반환값: 종가(원) × 거래량(주) = 거래대금(원)
앱 내부 변환: 원 ÷ 100,000,000 = 억원
기본 필터: 20일 평균 거래대금 ≥ 30억원
```

### RSI 계산

```python
# Wilder's Smoothing Method (14일)
# RSI < 70: 과열 아님 → 통과
# RSI ≥ 70: 과열 → 제외
```

### 종합 점수

```
종합점수 (100점 만점) = 재무점수 (50) + 차트점수 (40) + 유동성점수 (10)
```

## 🐛 자주 발생하는 오류

### `IndexError: range object index out of range`
- **원인**: pykrx가 날짜를 자동 감지할 때 공휴일/주말 처리 실패
- **해결**: 앱이 자동으로 이전 영업일로 재시도합니다

### `데이터 부족 (N일)` 로그 출력
- **원인**: 상장 초기 종목 또는 거래 정지 종목
- **해결**: 자동 제외됩니다 (정상)

### 전종목 분석이 너무 오래 걸림
- **해결**: 테스트 모드로 먼저 확인 후, 실전 모드는 장 마감 후 실행 권장
